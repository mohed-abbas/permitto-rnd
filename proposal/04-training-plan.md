# Section 4: Training Plan

---

## 4.1 Training Approach

| Aspect | Detail |
|--------|--------|
| Method | Single comprehensive train-the-trainer session with client's team |
| Format | Live session (online or on-site) covering all roles, workflows, and system administration |
| Timing | Conducted during Sprint 6 (UAT phase), before go-live |
| Environment | Conducted on the staging environment with test data |
| Self-Service Materials | Dedicated in-app training & help page accessible by all registered users |

---

## 4.2 Training Session

| Aspect | Detail |
|--------|--------|
| Audience | Client's designated team (train-the-trainer) |
| Duration | 3-4 hours |
| Format | Live walkthrough + Q&A |

### Session Agenda

| # | Topic | Coverage |
|---|-------|---------|
| 1 | System Overview & Navigation | Login, dashboard, navigation, notification center, PWA installation, offline mode |
| 2 | User & Project Setup | Creating users across all 5 roles, project creation, zone management, user-project assignment |
| 3 | Profile Signature Setup | One-time signature drawing and saving for all approval roles |
| 4 | Template Builder | Creating permit templates, configuring field types, multi-step sections, risk assessment checklists, background color, versioning, preview, activate/deactivate |
| 5 | Template Modification | Editing existing templates, creating new versions, duplicating templates |
| 6 | Permit Request Flow (WPR) | Selecting template, filling multi-step form, auto-save, file upload, draft management, submitting for approval |
| 7 | 4-Level Approval Flow | WPI approval, PTW-PC approval, HSSE approval — one-click approve, requesting additional measures, rejection with feedback |
| 8 | SIMOPs Management | Zone overlap alerts, countersign process, withhold mechanism, SIMOPs checklist |
| 9 | Daily Re-Approval | Daily confirmation, bulk approval, suspension and reactivation |
| 10 | Condition Change & Extensions | Flagging condition changes, re-assessment, permit extension request and approval |
| 11 | Close-Out & Cancellation | Permit close-out by PTW-PC, cancellation, PTW Register updates |
| 12 | PTW Register | Searching, filtering, exporting permit records |
| 13 | PDF Generation | Viewing and downloading permit PDFs, offline PDF access |
| 14 | Q&A | Open questions from the client's team |

After this session, the client's trained team will be equipped to onboard all end users (WPIs, HSSE Officers, WPRs) independently.

---

## 4.3 In-App Training & Help Page

A dedicated **Training & Help** page will be built into the web application, accessible by all registered users from the main navigation.

| Feature | Description |
|---------|-------------|
| Role-Based Guides | Each user sees training content relevant to their role (WPR sees permit request guide, WPI sees approval guide, etc.) |
| Step-by-Step Walkthroughs | Visual guides with screenshots for every major workflow |
| Video Walkthroughs | Recorded screen recordings embedded in the help page — covering all workflows demonstrated in the training session |
| Quick Reference | Common actions per role — one-page summary format |
| FAQ Section | Frequently asked questions and troubleshooting steps |
| Searchable | Users can search for specific topics or actions |
| Always Accessible | Available from any page via a help icon in the navigation — no need to leave the app |

## 4.4 Training Materials Summary

| Material | Delivery Method |
|----------|----------------|
| Live train-the-trainer session | Online or on-site (1 session, 3-4 hours) |
| Role-based user guides | In-app training page |
| Step-by-step walkthroughs with screenshots | In-app training page |
| Video recordings of training session | In-app training page |
| Quick reference per role | In-app training page |
| FAQ & troubleshooting | In-app training page |

All training materials are digital, accessible within the application, and available to every user at any time — no external documents to manage or distribute.
